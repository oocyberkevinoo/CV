MegaMix Editor (2022)

Editeur de "maps" pour le jeu de rythme Project Diva MegaMix.
Il permet de créer des maps pour le nouveau mode de jeu "Mix Mode".

Objectifs

- Permettre l'ajout de nouvelles chansons au jeu sur Switch.
- Permettre la modification de chansons déjà présentes dans le jeu.
